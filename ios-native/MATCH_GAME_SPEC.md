# マッチゲーム 要件定義書

## 概要
英単語と日本語訳のペアをタップで結びつけるタイムアタック型マッチングゲーム。Quizlet Matchをベースに、Merkenの学習データを活用する。

---

## 1. ゲームフロー

### 1.1 スタート画面
- ゲーム名「マッチ」+ アイコン表示
- ベストタイム表示（記録がある場合）
- 「スタート」ボタン

### 1.2 プレイ画面

#### グリッドレイアウト
- **6ペア = 12枚のカード** を画面に配置
- 左列: 英語カード6枚 / 右列: 日本語カード6枚（ランダム順）
- カード配置: 3行×4列グリッド（英語と日本語が混在、位置はランダム）

#### タイマー
- **カウントアップ方式**（0:00からスタート）
- 画面上部に常時表示
- 小数点1桁まで表示（例: 12.3秒）

#### 操作
1. プレイヤーが1枚目のカードをタップ → **選択状態**（青ボーダーでハイライト）
2. 2枚目のカードをタップ → **マッチ判定**
   - **正解**: 両カードが縮小→フェードアウトで消える + 成功haptic
   - **不正解**: 両カードが赤く光る + シェイクアニメーション + **ペナルティ+1.0秒**加算 + エラーhaptic
3. 同じカードを再タップ → 選択解除
4. 既に消えたカードはタップ不可

#### ラウンド制
- 単語数 > 6 の場合、6ペアずつラウンド分割
- ラウンド間: 「次のラウンド」ボタン（0.5秒の短い休憩）
- 全ラウンドの合計タイム = 最終スコア
- 単語数 ≤ 6 の場合、1ラウンドで完結

### 1.3 結果画面
- **合計タイム**（大きく表示）
- ペナルティ内訳: 「+X.0秒 ペナルティ（Y回ミス）」
- 正解数 / 総ペア数
- ベストタイム更新時 → 🎉エフェクト + 「新記録！」表示
- 「もう一度」ボタン
- 「閉じる」ボタン

## 2. カードUI

### 2.1 カードデザイン
- サイズ: グリッドに合わせて自動調整（画面幅 ÷ 4 - padding）
- 角丸: 12px
- 背景: `MerkenTheme.surface`
- ボーダー: `MerkenTheme.border` 1.5px

### 2.2 カード状態
| 状態 | 外観 |
|------|------|
| 未選択 | デフォルト（白背景、グレーボーダー） |
| 選択中 | 青ボーダー（`MerkenTheme.accentBlue`）+ 軽いスケールアップ(1.05) |
| 正解マッチ | 緑背景 → 縮小(0→) + フェードアウト、0.3秒で消える |
| 不正解 | 赤ボーダー + シェイク → 0.4秒後にデフォルトに戻る |
| 消去済み | `opacity(0)` + タップ無効 |

### 2.3 カードテキスト
- 英語: そのまま表示
- 日本語: そのまま表示
- フォントサイズ: 文字数に応じて自動縮小（`minimumScaleFactor(0.6)`）
- 長い単語: 2行まで許容

---

## 3. アニメーション仕様

| イベント | アニメーション | duration |
|---------|--------------|----------|
| カード選択 | スケール 1.0→1.05 + 青ボーダー | `MerkenSpring.tap` |
| 正解マッチ | スケール 1.0→1.1→0 + opacity→0 | 0.3秒 |
| 不正解 | 水平シェイク(±8px, 3回) + 赤ボーダー | 0.4秒 |
| ペナルティ表示 | タイマー横に「+1.0」がフロートアップ→フェードアウト | 0.6秒 |
| ラウンドクリア | 残りカードなし→0.3秒待ち→結果or次ラウンド | 0.3秒 |

---

## 4. データ

### 4.1 入力
- `[Word]` 配列（プロジェクトの全単語）
- 各Wordから `english` と `japanese` を使用

### 4.2 ラウンド分割ロジック
```
let shuffled = words.shuffled()
let rounds = shuffled.chunked(into: 6)  // 6ペアずつ
```

### 4.3 永続化
- ベストタイム: `UserDefaults` にプロジェクトIDごとに保存
  - key: `matchGame_bestTime_\(projectId)`

## 7. エッジケース

| ケース | 対応 |
|--------|------|
| 単語数 < 4 | マッチ非表示（最低4ペア必要） |
| 単語数 4-5 | 1ラウンド、ペア数 = 単語数 |
| 単語数 6+ | 6ペアずつラウンド分割 |
| 日本語訳が重複 | 重複する単語をラウンド内で避ける |
| 画面回転 | グリッドが自動リフロー |

---

## 8. 使用する既存コンポーネント

- `AppBackground` — 背景
- `MerkenTheme` — 色、フォント
- `MerkenSpring` — アニメーション
- `MerkenHaptic` — 触覚フィードバック
- `SolidCard` — 不要（カスタムグリッドカード）
