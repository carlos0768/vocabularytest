### Step 3: WordListView - Sort by Addition Date + Thick Dividers
In `WordListView.swift`, modify the word list display:

1. Sort `filteredWords` by `createdAt` descending (newest added first)
2. Group words by their `createdAt` date (same calendar day = same group)
3. Between groups from different dates, show a **3pt thick divider** (use `.frame(height: 3)` with `MerkenTheme.border` fill)
4. Within same-date groups, keep the existing 1px thin divider

Implementation approach for the grouped display:
```swift
// In the word list body, replace the simple ForEach with grouped display
let grouped = Dictionary(grouping: filteredWords) { word in
    Calendar.current.startOfDay(for: word.createdAt)
}
let sortedDates = grouped.keys.sorted(by: >)

ForEach(Array(sortedDates.enumerated()), id: \.element) { dateIndex, date in
    let wordsForDate = grouped[date]!
    
    if dateIndex > 0 {
        // Thick divider between date groups
        Rectangle()
            .fill(MerkenTheme.border)
            .frame(height: 3)
            .frame(maxWidth: .infinity)
    }
    
    ForEach(wordsForDate) { word in
        wordRow(word)
        dividerLine  // thin 1px divider within group
    }
}
```
