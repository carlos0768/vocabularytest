# デザイン修正指示（重要）

## 現状の問題

現在のデザインは要件定義書の意図から大きく外れています。以下の問題があります：

- グラデーション背景が使用されている → **禁止**
- グラスモーフィズム（半透明・ブラー効果）が使用されている → **禁止**
- ダークモード風になっている → **ライトモードで実装**
- 派手な視覚効果が追加されている → **不要**

これは「AI感が強い」デザインの典型例であり、ユーザーが求めている「落ち着いた・集中できる」学習アプリの方向性と真逆です。

---

## 絶対に守るべき禁止事項

以下は**絶対に使用しないでください**：

1. **グラデーション背景** - 背景は単色のみ
2. **グラスモーフィズム** - backdrop-filter: blur() は使用禁止
3. **半透明のカード** - カードの背景は不透明な白 `#ffffff`
4. **ダークモード配色** - ライトモードのみ
5. **オーロラ風・ネオン風の光エフェクト** - 一切不要
6. **box-shadowの過度な使用** - 影は極めて控えめに

## 正しいデザインの方向性

### 背景
```css
/* NG */
background: linear-gradient(135deg, #1a1a2e, #16213e, #0f3460);
background: radial-gradient(circle at 30% 70%, rgba(255, 0, 150, 0.3), transparent);

/* OK */
background-color: #fafaf9;
```

### カード
```css
/* NG */
background: rgba(255, 255, 255, 0.1);
backdrop-filter: blur(10px);
border: 1px solid rgba(255, 255, 255, 0.2);

/* OK */
background-color: #ffffff;
border: 1px solid #e7e5e4;
border-radius: 12px;
```

### シャドウ
```css
/* NG */
box-shadow: 0 8px 32px rgba(0, 0, 0, 0.3);

/* OK */
box-shadow: 0 1px 3px rgba(28, 25, 23, 0.05);
```

## 目指すべき参考イメージ

- **Medium** - 白背景、シンプルなカード、読みやすさ重視
- **Bear** - ウォームな配色、ミニマル、余白を活かす
- **Notion** - クリーンで機能的、装飾を排除

これらに共通するのは「**余計なものを足さない**」という思想です。

## 修正手順

1. **まず背景を修正** - 全ページの背景を `#fafaf9` の単色に変更
2. **カードを修正** - 不透明な白 `#ffffff` + 薄いボーダーに変更
3. **グラデーション・ブラーを全削除** - 検索して全て除去
4. **テキスト色を修正** - ダーク用の白文字から `#1c1917` に変更
5. **ヘッダーを修正** - シンプルな白背景 + 薄いボーダーに

## 最終確認

修正後、以下を確認してください：

- [ ] 背景にグラデーションがないか
- [ ] `backdrop-filter` が使われていないか
- [ ] カードが不透明な白になっているか
- [ ] テキストが読みやすい濃い色になっているか
- [ ] 全体的に「Medium」や「Notion」のような落ち着いた印象か

派手さは一切不要です。**シンプルで読みやすく、学習に集中できるUI**を目指してください。
