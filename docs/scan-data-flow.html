2つの違い： Free は /api/extract の1本のリクエスト内で Gemini（OCR抽出・訳語・例文）と Supabase 呼び出しがすべて同期的に完結し、レスポンス本文がそのまま結果データになります。 Pro は /api/scan-jobs/create が jobId だけ返して after() で非同期処理し、クライアントはポーリング + Supabase 再取得で結果を受け取る分離構造です。 ※ このスキャン経路で OpenAI は呼ばれません（例文生成も含め全て Gemini。設定キー名 openai は実体 gemini）。
