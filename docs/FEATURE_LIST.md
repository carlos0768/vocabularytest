# WordSnap 機能一覧

## 実装済み

### 全員
- 画像スキャン → AI単語抽出
- 4択クイズ（英→日）
- プロジェクト（単語帳）管理
- 学習統計（今日の問題数、正答率、習得数、連続日数）

### Pro限定
- スキャン無制限（Free: 5回/日）
- 単語数無制限（Free: 50語）
- クラウド保存・デバイス間同期
