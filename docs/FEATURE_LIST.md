# WordSnap 機能一覧

## 実装済み

### 全員
- 画像スキャン → AI単語抽出
- 4択クイズ（英→日）
- プロジェクト（単語帳）管理
- 学習統計（今日の問題数、正答率、習得数、連続日数）
- 苦手マーク（付ける・一覧を見る）

#### 苦手マーク機能 - 実装完了（2025-01-24）
- クイズ画面でハートアイコンをタップ → 苦手マーク ON/OFF
- プロジェクト詳細画面で苦手フィルター表示
- ホーム画面に「苦手な単語」リンク追加（全プロジェクト横断）
- `/favorites` ページで苦手単語一覧表示
- Supabase マイグレーション適用済み（`is_favorite` カラム追加）

### Pro限定
- スキャン無制限（Free: 5回/日）
- 単語数無制限（Free: 50語）
- クラウド保存・デバイス間同期
- 忘却曲線復習
- 例文学習
- 苦手だけクイズ

#### 例文学習機能 - 実装完了（2026-01-24）
- スキャン時にAIが各単語の例文（英語＋日本語訳）を自動生成
- プロジェクト詳細画面で単語ごとに例文を表示
- クイズ回答後に例文を表示して理解を深める
- Proユーザーのみ自動適用（Freeユーザーは従来通り単語のみ抽出）

#### 忘却曲線復習機能 - 実装完了（2026-01-24）
- SM-2アルゴリズムによる最適な復習タイミング計算
- 正解→復習間隔延長（1日→6日→間隔×難易度係数）
- 不正解→翌日再復習
- プロジェクト詳細画面に「復習」ボタン追加（復習必要数バッジ表示）
- `/review/[projectId]` ページで復習モード
- Supabase マイグレーション適用済み（`last_reviewed_at`, `next_review_at`, `ease_factor`, `interval_days`, `repetition` カラム追加）

#### 苦手だけクイズ機能 - 実装完了（2025-01-24）
- プロジェクト詳細画面に「苦手だけ」ボタン追加（苦手マーク数バッジ表示）
- `/quiz/[projectId]/favorites` ページで苦手単語のみのクイズモード
- 赤テーマのUI（通常クイズと視覚的に区別）
- Proユーザー限定機能

### アカウント登録で解放（Free/Pro共通）
- 単語帳シェア

#### 単語帳シェア機能 - 実装完了（2026-01-24）
- プロジェクト詳細画面のヘッダーに共有ボタン追加
- タップで共有リンク生成＆クリップボードにコピー
- 共有URLにアクセス → ログイン必須（未ログインはリダイレクト）
- 共有された単語帳を閲覧可能（読み取り専用）
- 「自分の単語帳に追加」ボタンでインポート（コピー作成）
- Supabase マイグレーション適用済み（`share_id` カラム追加、RLS設定）
