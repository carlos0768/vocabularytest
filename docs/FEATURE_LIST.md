# WordSnap 機能一覧

## 実装済み

### 全員
- 画像スキャン → AI単語抽出
- 4択クイズ（英→日）
- プロジェクト（単語帳）管理
- 学習統計（今日の問題数、正答率、習得数、連続日数）
- 苦手マーク（付ける・一覧を見る）

#### 苦手マーク機能 - 実装完了（2025-01-24）
- クイズ画面でハートアイコンをタップ → 苦手マーク ON/OFF
- プロジェクト詳細画面で苦手フィルター表示
- ホーム画面に「苦手な単語」リンク追加（全プロジェクト横断）
- `/favorites` ページで苦手単語一覧表示
- Supabase マイグレーション適用済み（`is_favorite` カラム追加）

### Pro限定
- スキャン無制限（Free: 5回/日）
- 単語数無制限（Free: 50語）
- クラウド保存・デバイス間同期
