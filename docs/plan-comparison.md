## スキャンモード（全プラン共通）

| モード | 説明 |
|--------|------|
| `all` | 全単語抽出 |
| `circled` | 丸囲みの単語のみ |
| `highlighted` | マーカー部分のみ |
| `eiken` | 英検レベルフィルタ |
| `idiom` | イディオム抽出 |
| `wrong` | 間違えた単語のみ |
