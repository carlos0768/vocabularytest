## 備考

- Gemini 2.5 Flash の画像トークン数は解像度に依存（圧縮後 ~1,000トークンで概算）
- 1スキャンあたりの抽出単語数は15語で概算（実際は5〜30語程度）
- GPT-4o-miniのバッチ処理により、単語数増でもAPI呼び出し回数は変わらない
- maxOutputTokensは上限設定であり、実際の出力はそれより大幅に少ない
- クイズ導線のコストは通常¥0（ディストラクターはスキャン時に生成済み）
- Word Insightsは単語ごとに逐次処理（1単語1 API呼び出し）
- Embeddings同期のコストは実質無視できるレベル
