# Maintenance Archive

ここは、保守性向上工事で完了した作業ログや古い引き継ぎを退避する場所です。

## 使い方

- 現在進行中の情報は `docs/maintenance/` 直下に置く。
- 完了した作業ログや古くなったROADMAPは、日付つきディレクトリに移す。
- 恒久的に残すべき知識は、ここに置きっぱなしにせず、正式docsへ昇格させる。

例:

```txt
docs/maintenance/_archive/
  2026-prelaunch-cleanup/
    AI_HANDOFF.md
    TASKS.md
    notes.md
```
